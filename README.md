# High-Performance Distributed Computing

> **Track:** NVIDIA HPC & Developer Upskilling (Track 2)
> **Course 6 of 8**

Implement multi-GPU and multi-node distributed training and inference with MPI, NCCL, and fault-tolerant patterns.

---

## Modules

### Module 1: MPI Fundamentals & Collective Operations

**Learning Objectives:**
- Run MPI jobs across nodes
- Use AllReduce for gradient aggregation
- Debug rank communication issues

### Module 2: NCCL for Multi-GPU Communication

**Learning Objectives:**
- Initialize NCCL communicators
- Implement AllGather and Broadcast
- Profile NCCL bandwidth utilization

### Module 3: Distributed Training Patterns

**Learning Objectives:**
- Implement data parallelism with DDP
- Apply pipeline and tensor parallelism
- Scale from 1 to 8 GPUs

### Module 4: Fault Tolerance & Checkpointing

**Learning Objectives:**
- Save training checkpoints at intervals
- Resume interrupted training jobs
- Implement elastic training with recovery

### Module 5: Network Topology Optimization

**Learning Objectives:**
- Configure InfiniBand vs. Ethernet
- Measure all-reduce bandwidth
- Tune topology-aware communication
